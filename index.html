Web Developer

Building for the web — one well-crafted project at a time.

I'm a web developer working at GetUWired in Dahlonega, GA. My work spans front-end and back-end development, with a focus on writing code that is clean, maintainable, and built on solid semantic foundations.

I care about the details that most people don't see — proper HTML structure, accessible markup, meaningful structured data, and performance that doesn't make users wait. Those foundations separate a site that just looks good from one that actually works well for everyone.

Outside of GetUWired, I build websites for clients I genuinely believe in — including Outdoor Bills for musician Bill Leroy of the Savannah Bananas, and Kentucky Pikes for Pi Kappa Alpha at the University of Kentucky.

Good code is written for the next developer. Great code is written for the user who never sees it.

What I do

Front-End Development

I build interfaces using semantic HTML5, CSS3, and JavaScript. Every project I ship is validated, accessible, and structured with care. I rely on Flexbox for layouts and write CSS that scales cleanly without relying on hacks or heavy frameworks.

Back-End & Full Stack

My real interest is in how systems connect. Backend logic, APIs, data flow, and building reliable behavior when real users interact with real inputs — that's where I spend a lot of my energy. At GetUWired I work across the stack, which has sharpened both sides of my skill set.
